For Vercel deployment:

Set these environment variables in Vercel:
NODE_ENV=production
VITE_API_URL=https://your-vercel-url.vercel.app
All SUPABASE_* credentials

On Vercel, you can update environment variables here:

Go to your project on vercel.com
Click Settings (top navigation)
Select "Environment Variables" (left sidebar)
Add your variables:
Click "Add New"
Enter the variable name (e.g., VITE_API_URL)
Enter the value
Select which environment(s): Production, Preview, Development
Click "Save"
For your shipping app, add these:

Variable	Value	Environment
NODE_ENV	production	Production
VITE_API_URL	https://your-vercel-url.vercel.app	Production, Preview
SUPABASE_USER	Your Supabase user	Production
SUPABASE_HOST	Your Supabase host	Production
SUPABASE_DATABASE	Your Supabase database	Production
SUPABASE_PASSWORD	Your Supabase password	Production
SUPABASE_PORT	5432	Production
JWT_SECRET	Your JWT secret	Production
STRIPE_SECRET_KEY	Your Stripe key	Production
VITE_EASYSHIP_SAND	Your Easyship key	Production
Pro tip: You can copy your .env values and paste them directly into Vercel's environment variables form.
